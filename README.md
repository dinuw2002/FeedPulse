🚀 FeedPulse: AI-Powered Feedback Intelligence

FeedPulse is a full-stack MERN application designed to bridge the gap between user feedback and actionable product insights. Using Gemini 2.5 Flash AI, it automatically analyzes, categorizes, and prioritizes incoming feedback in real-time.

🌟 Key Features
AI-Driven Analysis: Automatic sentiment detection, 1-sentence summaries, and priority scoring (1-10) using Google Gemini REST API.

Smart Dashboard: Admins can search, filter by category/status, and track live stats (Average Priority, Top Tags).

Weekly AI Reports: On-demand "Top 3 Themes" summary generated from the last 7 days of data.

Self-Healing AI: Manual "Re-trigger" action for admins to re-analyze any submission.

Secure Admin Access: JWT-based authentication with bcrypt password hashing and protected API routes.

🛠️ Tech Stack
Frontend: Next.js 14+, Tailwind CSS, TypeScript, Lucide React.

Backend: Node.js, Express.js, TypeScript.

Database: MongoDB with Mongoose (ODM).

AI Integration: Google Gemini 2.5 Flash REST API.

⚙️ Environment Variables
Create a .env file in the backend folder:

PORT=5000
MONGO_URI="your_mongodb_connection_string"
JWT_SECRET=your_super_secret_key
GEMINI_API_KEY=your_google_ai_studio_api_key
ADMIN_EMAIL="your_admin_email"
ADMIN_PASSWORD="your_admin_password"

Create a .env.local file in the backend folder:

NEXT_PUBLIC_API_URL=your_back_end_running_url

🏃 How to Run Locally

1. Clone the Repository

git clone https://github.com/your-username/feedpulse.git
cd feedpulse

2. Setup Backend

cd backend
npm install
npm run dev

3. Setup Frontend

cd frontend
npm install
npm run dev

The app will be running at http://localhost:3000
